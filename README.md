📌 AI Telegram Bot (Yandex LLM + Image Generation)

Telegram-бот с интеграцией LLM API (Yandex) и генерацией изображений, реализующий мультимодальное взаимодействие с пользователем (текст + изображения).

Возможности:

- Генерация текстов через LLM (Yandex API)
- Генерация изображений по запросу
- Базовые команды Telegram-бота
- Асинхронная обработка сообщений
- Интеграция с внешними API


🚀 Запуск проекта
1. Клонирование
git clone https://github.com/Victordydka/telegrem_bot_API_AI.git
cd telegrem_bot_API_AI
2. Виртуальное окружение
python -m venv venv
Активация:

Linux / Mac:

source venv/bin/activate

Windows:

venv\Scripts\activate

3. Установка зависимостей
pip install --upgrade pip
pip install -r requirements.txt

4. Запуск
python main.py
